# Calcme

  Api da aplicação CALCME. A aplicação resume-se a envio de dados de um usuário com o email, nome e telephone e salvar isso no banco de dados.
  
## Requisitos
 - Angular 10
 - Node.js 12
## Iniciando a aplicação
  - Rode no terminal
```
npm install
```
  - Depois
 ```
ng serve --open
```
